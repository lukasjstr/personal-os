Personal OS – Die definitive Spezifikation
Was das hier ist
Ein System das dein Leben operativ führt. Du gibst alles rein. Es sagt dir was du als nächstes tun sollst. Immer. Ohne dass du nachdenken musst.

Das Kernprinzip

text


DU DENKST NICHT MEHR.
DAS SYSTEM DENKT FÜR DICH.

Du teilst ALLES:
├── "Ich brauch Milch und Eier"
├── "Bankdrücken 30kg x8"
├── "Landingpage ist fertig"  
├── "Ich fühl mich heute schlapp"
├── "Mittwoch 14 Uhr Zahnarzt"
├── "Ich sollte mal meine Steuern machen"
├── "3 Flaschen Wasser getrunken"
├── [Foto von Gewichten]
├── [Sprachnachricht mit Idee]
└── Alles. Wirklich alles.

DAS SYSTEM:
├── Versteht was es ist
├── Ordnet es ein (Ziel? Task? Einkauf? Log? Idee?)
├── Speichert es am richtigen Ort
├── Leitet Aktionen ab
├── Erinnert dich proaktiv
├── Sagt dir was als NÄCHSTES kommt
├── Pusht dich wenn du nachlässt
└── Feiert wenn du lieferst

DU MACHST NUR NOCH:
├── Input geben (Telegram)
├── Ausführen was es dir sagt
└── Das wars.

Proaktivität – Das Herzstück
Das unterscheidet es von jedem Task-Manager. Das System wartet nicht. Es kommt zu dir.

text


══════════════════════════════════════════
WIE PROAKTIVITÄT FUNKTIONIERT
══════════════════════════════════════════

Jedes Ziel, jeder Task, jede Routine erzeugt
AUTOMATISCH Erinnerungen und Push-Nachrichten.

BEISPIEL: "3 Liter Wasser am Tag"
System erstellt automatisch:
├── 09:00 Push: "💧 Erste Flasche Wasser!"
├── 13:00 Push: "💧 Zweite Flasche. Stand: X/3L"
├── 17:00 Push: "💧 Letzte Flasche. Noch X.XL übrig"
└── Wenn User loggt → Anpassung in Echtzeit

BEISPIEL: "4x Training pro Woche"
System weiß: Es ist Donnerstag, erst 1x trainiert
├── 08:00 Push: "⚠️ Erst 1/4 Training diese Woche.
│   Heute und morgen noch je ein Training nötig.
│   Wann willst du heute trainieren?"
├── User sagt "18 Uhr"
├── → Kalender-Eintrag 18:00 Training
├── 17:30 Push: "💪 Training in 30 Minuten!"
└── Nach Training-Log: "2/4 ✅ Noch 2 bis Freitag."

BEISPIEL: "Einkaufen"
User hat im Lauf der Woche gesagt:
├── "Milch kaufen"
├── "Eier brauch ich auch"
├── "Proteinpulver ist leer"
System:
├── Sammelt alles in einer Einkaufsliste
├── Samstag 10:00: "🛒 Einkaufsliste:
│   ☐ Milch
│   ☐ Eier  
│   ☐ Proteinpulver
│   Gehst du heute einkaufen?"
└── User: "ja" → "Soll ich einen Kalender-Eintrag
    machen? Wann?"

BEISPIEL: "Landingpage bauen"  
System hat 5 Tasks, 2 erledigt:
├── Morgen-Brief: "Nächster Schritt Landingpage:
│   Testimonials sammeln. 45min Fokus-Block?"
├── Wenn 3 Tage keine Aktivität:
│   "⚠️ Landingpage seit 3 Tagen keine Action.
│   Deadline in 5 Tagen. Heute 1 Stunde dran?"
├── Wenn Task erledigt:
│   "✅ Testimonials done! Nächster Schritt:
│   CTA testen. Soll ich einen Zeitblock einplanen?"
└── Wenn alles fertig:
    "🎉 Landingpage KOMPLETT! Key Result erreicht!
    Nächste Priorität in Business: LinkedIn Content"

DAS PRINZIP:
Jeder Eintrag im System erzeugt einen 
Reminder-Stream der den User durch die 
Ausführung FÜHRT.

Nicht: "Hier sind deine Tasks."
Sondern: "Tu JETZT das. Dann das. Dann das."

Toggle-System

text


USER-EINSTELLUNGEN (per Telegram steuerbar):

"Prioritäten an" / "Prioritäten aus"
├── AN: Morgen-Brief mit Top 3, automatische 
│       Priorisierung, Tages-Führung
└── AUS: System speichert alles, erinnert an
         Routinen/Termine, aber keine Priorisierung

"Review an" / "Review aus"  
├── AN: Abend-Review mit Tages-Score
└── AUS: Kein Abend-Review

"Proaktiv an" / "Proaktiv aus"
├── AN: System pusht aktiv (Wasser-Reminder,
│       Training-Nudges, Task-Erinnerungen)
└── AUS: System antwortet nur wenn du schreibst

"Reflection" → Immer Sonntag Abend Einladung
├── User wird gefragt "Willst du reflektieren?"
├── JA → geführte Session startet
├── NEIN / keine Antwort → wird übersprungen
├── "Reflection aus" → Wird gar nicht erst gefragt
└── "Reflection an" → Wieder aktiv

BEFEHLE:
/settings → Zeigt alle Toggles
/toggle priorities → An/Aus
/toggle review → An/Aus  
/toggle proactive → An/Aus
/toggle reflection → An/Aus
/times → Zeigt eingestellte Zeiten
/times morning 07:00 → Ändert Morning Brief Zeit
/times evening 21:30 → Ändert Evening Review Zeit

INPUT-KATEGORIEN (automatisch erkannt):

1. ZIELE & WÜNSCHE
   "Ich will abnehmen"
   "Ich muss mehr Sport machen"
   "Dieses Jahr will ich 100k Umsatz"
   → Objective + Key Results vorschlagen

2. FORTSCHRITT & LOGS
   "Bankdrücken 30kg x8"
   "2 Flaschen Wasser"
   "Landingpage Hero Section fertig"
   → Log erstellen, Progress updaten

3. TASKS & AUFGABEN
   "Ich muss die Steuererklärung machen"
   "Testimonials sammeln"
   "Website updaten"
   → Task erstellen, einordnen

4. ERLEDIGUNGEN
   "Fertig" / "Done" / "Erledigt" / "✅"
   "Landingpage ist live"
   "Training war gut"
   → Task/Routine abhaken, nächsten Schritt zeigen

5. TERMINE & KALENDER
   "Mittwoch 14 Uhr Zahnarzt"
   "Nächsten Montag Meeting mit Team"
   → CalendarEvent erstellen + Reminder planen

6. EINKAUFEN & LISTEN
   "Milch kaufen"
   "Brauch noch Proteinpulver"  
   → Einkaufsliste (spezieller Task-Typ)

7. ROUTINEN
   "Jeden Morgen meditieren"
   "3x pro Woche laufen"
   "Sonntag Meal Prep"
   → Routine erstellen + Reminder-Schedule

8. GEDANKEN & BRAIN DUMPS
   "Ich glaub ich sollte mal über Podcast nachdenken"
   "Interessante Idee: App für Hundbesitzer"
   → Brain Dump speichern, später einordnen

9. BEFINDLICHKEIT
   "Bin heute müde"
   "Mega produktiv heute"
   "Fühle mich überfordert"
   → Mood-Log, Pattern-Erkennung

10. EINSTELLUNGEN
    "Prioritäten aus"
    "Erinner mich nicht mehr abends"
    /settings, /toggle, /times
    → Settings ändern

11. FRAGEN ANS SYSTEM
    "Was steht heute an?"
    "Wie läuft mein Fitness-Ziel?"
    "Was hab ich diese Woche geschafft?"
    → Kontext-basierte Auskunft

Das System denkt IMMER einen Schritt voraus.
Es gibt nie ein Vakuum nach einer Erledigung.

async def process_update(update_data: dict):
    try:
        # 1. Parse update
        message = extract_message(update_data)
        if not message:
            return
        
        telegram_id = message["from"]["id"]
        username = message["from"].get("username")
        first_name = message["from"].get("first_name")
        
        # 2. Check for commands first
        if message.get("text", "").startswith("/"):
            await handle_command(telegram_id, message["text"])
            return
        
        # 3. Get or create user
        async with get_session() as session:
            user = await get_or_create_user(
                session, telegram_id, username, first_name
            )
            
            # 4. Determine input type
            input_text = None
            source = "text"
            
            if "text" in message:
                input_text = message["text"]
            elif "photo" in message:
                input_text = message.get("caption", "[Foto gesendet]")
                source = "image"
                # Phase 2: Vision API
            elif "voice" in message:
                input_text = "[Sprachnachricht – Voice-to-Text kommt in Phase 2]"
                source = "voice"
            
            if not input_text:
                return
            
            # 5. Save user message
            await save_conversation(
                session, user.id, "user", input_text,
                session_date=date.today()
            )
            
            # 6. Build context
            context = await build_context(session, user.id)
            
            # 7. Get conversation history (last 10)
            history = await get_recent_conversations(
                session, user.id, limit=10
            )
            
            # 8. Send to Haiku
            ai_client = get_ai_client()
            result = await ai_client.process_message(
                user_message=input_text,
                context=context,
                conversation_history=history,
                user_id=user.id,
                session=session
            )
            
            # 9. Send response to user
            if result["text"]:
                await send_message(
                    telegram_id, result["text"]
                )
            
            # 10. Save assistant response
            await save_conversation(
                session, user.id, "assistant", 
                result["text"],
                tool_calls=result.get("tool_calls"),
                tokens_used=result.get("tokens_used"),
                session_date=date.today()
            )
            
            await session.commit()
    
    except Exception as e:
        logger.error(f"Error processing update: {e}", exc_info=True)
        try:
            await send_message(
                telegram_id,
                "⚠️ Kurze Störung. Versuch es gleich nochmal."
            )
        except:
            pass

def extract_message(update_data: dict) -> dict | None:
    """Extract message from update, handle both 
    message and edited_message."""
    return update_data.get("message") or update_data.get("edited_message")

async def get_or_create_user(session, telegram_id, username, first_name):
    """Get existing user or create new one with default settings."""
    import uuid
    result = await session.execute(
        select(User).where(User.telegram_id == telegram_id)
    )
    user = result.scalar_one_or_none()
    if not user:
        user = User(
            telegram_id=telegram_id,
            telegram_username=username,
            first_name=first_name,
            settings={
                "priorities_enabled": True,
                "review_enabled": True,
                "proactive_enabled": True,
                "reflection_enabled": True,
                "morning_brief_time": "06:30",
                "evening_review_time": "21:00",
                "weekly_reflection_day": "sunday",
                "weekly_reflection_time": "19:00",
                "shopping_reminder_day": "saturday",
                "shopping_reminder_time": "10:00",
                "ical_token": str(uuid.uuid4())
            }
        )
        session.add(user)
        await session.flush()
        logger.info(f"New user created: {telegram_id}")
    return user

async def send_message(chat_id: int, text: str, 
                       parse_mode: str = "Markdown"):
    url = f"https://api.telegram.org/bot{token}/sendMessage"
    # Split long messages (Telegram limit 4096 chars)
    chunks = split_message(text, 4096)
    async with httpx.AsyncClient() as client:
        for chunk in chunks:
            payload = {
                "chat_id": chat_id,
                "text": chunk,
                "parse_mode": parse_mode
            }
            resp = await client.post(url, json=payload)
            if resp.status_code != 200:
                # Retry without parse_mode (markdown might be broken)
                payload["parse_mode"] = None
                await client.post(url, json=payload)

class AIClient:
    def __init__(self):
        self.client = anthropic.Anthropic(
            api_key=settings.ANTHROPIC_API_KEY
        )
    
    async def process_message(
        self,
        user_message: str,
        context: str,
        conversation_history: list[dict],
        user_id: int,
        session: AsyncSession
    ) -> dict:
        """
        Send message to Haiku, handle tool call loop.
        Returns {"text": str, "tool_calls": list, "tokens_used": int}
        """
        
        system_prompt = SYSTEM_PROMPT.format(context=context)
        
        # Build messages from conversation history
        messages = []
        for conv in conversation_history:
            messages.append({
                "role": conv["role"],
                "content": conv["content"]
            })
        messages.append({
            "role": "user",
            "content": user_message
        })
        
        all_tool_calls = []
        total_tokens = 0
        max_iterations = 5
        
        for iteration in range(max_iterations):
            # Call Haiku
            response = self.client.messages.create(
                model="claude-haiku-4-20250514",
                max_tokens=1024,
                temperature=0.3,
                system=system_prompt,
                messages=messages,
                tools=TOOLS
            )
            
            total_tokens += response.usage.input_tokens + response.usage.output_tokens
            
            # Check if there are tool calls
            tool_use_blocks = [
                b for b in response.content 
                if b.type == "tool_use"
            ]
            text_blocks = [
                b.text for b in response.content 
                if b.type == "text"
            ]
            
            if not tool_use_blocks:
                # No more tool calls, we're done
                return {
                    "text": "\n".join(text_blocks),
                    "tool_calls": all_tool_calls,
                    "tokens_used": total_tokens
                }
            
            # Execute tool calls
            tool_results = []
            for tool_block in tool_use_blocks:
                result = await execute_tool(
                    tool_block.name,
                    tool_block.input,
                    user_id,
                    session
                )
                all_tool_calls.append({
                    "name": tool_block.name,
                    "input": tool_block.input,
                    "result": result
                })
                tool_results.append({
                    "type": "tool_result",
                    "tool_use_id": tool_block.id,
                    "content": json.dumps(result)
                })
            
            # Add assistant response and tool results to messages
            messages.append({
                "role": "assistant",
                "content": response.content
            })
            messages.append({
                "role": "user",
                "content": tool_results
            })
        
        # Max iterations reached
        final_text = "\n".join(text_blocks) if text_blocks else "Verarbeitung abgeschlossen."
        return {
            "text": final_text,
            "tool_calls": all_tool_calls,
            "tokens_used": total_tokens
        }

async def build_context(session: AsyncSession, user_id: int) -> str:
    """Build compact context string for Haiku.
    Max ~1500 tokens. Only the most relevant info."""
    
    parts = []
    
    # 1. Active Objectives with Key Results
    objectives = await get_active_objectives_with_kr(session, user_id)
    if objectives:
        parts.append("AKTIVE ZIELE:")
        for obj in objectives:
            parts.append(f"  [{obj.category.upper()}] {obj.title} (ID:{obj.id})")
            for kr in obj.key_results:
                progress = ""
                if kr.target_value:
                    pct = (kr.current_value / kr.target_value * 100) if kr.target_value else 0
                    progress = f" → {kr.current_value}/{kr.target_value} {kr.unit or ''} ({pct:.0f}%)"
                parts.append(f"    - KR(ID:{kr.id}): {kr.title}{progress} [{kr.frequency}]")
    
    # 2. Open Tasks (top 5 by priority)
    tasks = await get_prioritized_tasks(session, user_id, limit=5)
    if tasks:
        parts.append("\nOFFENE TASKS:")
        for t in tasks:
            due = f" (fällig: {t.due_date})" if t.due_date else ""
            cat = f" [{t.category}]" if t.category != "general" else ""
            parts.append(f"  Task(ID:{t.id}) [P{t.priority}]{cat}: {t.title}{due}")
    
    # 3. Shopping items (open)
    shopping = await get_open_shopping_items(session, user_id)
    if shopping:
        parts.append(f"\nEINKAUFSLISTE ({len(shopping)} Items):")
        for s in shopping:
            parts.append(f"  ☐ {s.title}")
    
    # 4. Today's routines
    routines = await get_todays_routines_with_status(session, user_id)
    if routines:
        parts.append("\nROUTINEN HEUTE:")
        for r, done in routines:
            status = "✅" if done else "☐"
            parts.append(f"  {status} {r.title} (ID:{r.id})")
    
    # 5. Today's calendar events
    events = await get_todays_events(session, user_id)
    if events:
        parts.append("\nTERMINE HEUTE:")
        for e in events:
            time_str = e.start_time.strftime("%H:%M")
            parts.append(f"  {time_str} {e.title}")
    
    # 6. Recent logs (last 5)
    logs = await get_recent_logs(session, user_id, limit=5)
    if logs:
        parts.append("\nLETZTE AKTIVITÄT:")
        for log in logs:
            time_str = log.created_at.strftime("%d.%m %H:%M")
            parts.append(f"  {time_str}: [{log.log_type}] {summarize_log(log)}")
    
    # 7. Mood trend
    moods = await get_mood_trend(session, user_id, days=7)
    if moods:
        parts.append(f"\nMOOD (7 Tage): {', '.join(str(m) for m in moods)}")
    
    # 8. User settings/toggles
    user = await get_user(session, user_id)
    s = user.settings
    parts.append(f"\nSETTINGS: Priorities={'AN' if s.get('priorities_enabled') else 'AUS'}, "
                 f"Review={'AN' if s.get('review_enabled') else 'AUS'}, "
                 f"Proaktiv={'AN' if s.get('proactive_enabled') else 'AUS'}")
    
    return "\n".join(parts)

def summarize_log(log) -> str:
    """Create one-line summary of a log entry."""
    data = log.data or {}
    if log.log_type == "workout":
        return f"{data.get('exercise')} {data.get('weight')}kg×{data.get('reps')}×{data.get('sets', 1)}"
    elif log.log_type == "water":
        return f"{data.get('amount')}L Wasser"
    elif log.log_type == "mood":
        return f"Mood {data.get('score')}/10"
    elif log.log_type == "progress":
        return data.get("description", str(data))
    else:
        return str(data)[:50]

async def execute_tool(
    tool_name: str, 
    tool_input: dict, 
    user_id: int, 
    session: AsyncSession
) -> dict:
    """Execute a tool call and return the result."""
    
    try:
        match tool_name:
            case "create_objective":
                obj = await create_objective(
                    session, user_id, **tool_input
                )
                return {"success": True, "objective_id": obj.id, 
                        "message": f"Objective '{obj.title}' erstellt"}
            
            case "create_key_result":
                kr = await create_key_result(
                    session, user_id=user_id, **tool_input
                )
                return {"success": True, "key_result_id": kr.id,
                        "message": f"Key Result '{kr.title}' erstellt"}
            
            case "create_task":
                task = await create_task(
                    session, user_id, **tool_input
                )
                return {"success": True, "task_id": task.id,
                        "message": f"Task '{task.title}' erstellt"}
            
            case "complete_task":
                task = await complete_task(
                    session, tool_input["task_id"]
                )
                # Find next task in same key result
                next_task = None
                if task.key_result_id:
                    next_task = await get_next_task_in_kr(
                        session, task.key_result_id
                    )
                return {
                    "success": True,
                    "completed": task.title,
                    "next_task": {
                        "id": next_task.id, 
                        "title": next_task.title
                    } if next_task else None
                }
            
            # ... implement ALL tools similarly
            
            case _:
                return {"success": False, 
                        "error": f"Unknown tool: {tool_name}"}
    
    except Exception as e:
        logger.error(f"Tool execution error: {tool_name}: {e}")
        return {"success": False, "error": str(e)}

ical_router = APIRouter()

@ical_router.get("/{user_token}.ics")
async def ical_feed(user_token: str):
    """Serve iCal feed for a user.
    URL can be subscribed in Google/Apple Calendar."""
    
    async with get_session() as session:
        # Find user by ical_token in settings
        user = await get_user_by_ical_token(session, user_token)
        if not user:
            raise HTTPException(404)
        
        ical_string = await generate_ical(session, user.id)
        
        return Response(
            content=ical_string,
            media_type="text/calendar",
            headers={
                "Content-Disposition": f"attachment; filename=personal-os.ics"
            }
        )
